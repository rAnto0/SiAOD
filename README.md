# Лабораторные работы по СиАОД

Репозиторий с лабораторными работами и курсовым проектом по дисциплине «Структуры и алгоритмы обработки данных».

Автор: **Балыко Владислав Васильевич**, группа **БФИ2201**.

## Содержимое

- `Lab1.ipynb` — сравнение сортировок (Шелла, турнирная, пузырьковая), замеры времени, графики.
- `Lab2.ipynb` и `data/Lab2/` — задачи на стек, очередь и дек: сортировка строк, шифрование/дешифрование, Ханойские башни и др.
- `Lab3.ipynb` и `data/Lab3/graph_matrix.txt` — графы, алгоритм Дейкстры, визуализация и сравнение производительности.
- `Lab4.ipynb` — алгоритмы поиска, хеш-таблицы и задача о восьми ферзях.
- `Lab5.ipynb` — KMP, упрощенный Бойер-Мур и решение `15-puzzle` алгоритмом A*.
- `Kurs/kurs.ipynb` и `Kurs/` — курсовая работа: анализ продаж из CSV, отчет и визуализация.

## Требования

- Python `>=3.11`
- Основные библиотеки:
  - `pandas`
  - `matplotlib`
  - `networkx`
- Для запуска ноутбуков дополнительно:
  - `jupyter`
  - `ipykernel`

## Быстрый старт

### Вариант 1: через `uv` (рекомендуется)

```bash
uv sync
uv sync --extra notebooks
uv run jupyter lab
```

### Вариант 2: через `venv` и `pip`

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .
pip install -e .[notebooks]
jupyter lab
```

## Примечания

- Данные для `Lab2.ipynb` и `Lab3.ipynb` находятся в папке `data/` (`data/Lab2/` и `data/Lab3/`).
- В `Kurs/kurs.ipynb` используются файлы `Kurs/sales.csv` и `Kurs/sales_report.csv`.
- Некоторые задания в ноутбуках ожидают ввод с клавиатуры.
